My Projects

Some of my recent work across various domains

BAS App

BAS Project is a web-based automation application that enables consumers to recharge prepaid gas meters and view transaction history, integrating NTT DATA Payment Gateway for seamless processing. It ensures efficient management and secure transactions, optimizing billing accuracy and reducing manual intervention. It provides scalable architecture, fast processing, and secure financial transactions, enhancing the user experience for both customers and service providers

Technologies: ASP.NET, .NET Core 8.0, CQRS, SOLID Principles

Captive CNG Station App

Captive CNG Station App is a web-based fuel automation system for appointments, EV recharging, and vehicle refueling. It supports Super Admin, Business Partners, Supervisors, Managers, and End Customers, enabling business registration, location setup, and communication device management. Customers can book timeslots, verified via PLC devices, MQTT protocol, and camera-based authentication using vehicle number and OTP.It ensures efficient scheduling and automated transactions

Technologies: ASP.NET, .NET Core 8.0, CQRS, SOLID Principles, and Onion Architecture, MQTT

Sports App

Sports App is a mobile-based application designed for sports clubs, enabling Admin, Coach, and Student roles to efficiently manage player profiles, tournaments, and events. It allows coaches to assess skills, provide performance ratings, and track athlete development over time. The app simplifies club operations by integrating match scheduling, student progress tracking, and tournament management in one streamlined platform. It ensures seamless interactions, real-time updates, and optimized user experience for all stakeholders..

Technologies: Flutter, .NET Core 3.1, Dapper, MSSQL

Enlight (LMS)

Enlight (LMS) is a multi-platform educational management system available on Web, Desktop, and Mobile. It streamlines school administration, including teacher/student management, class organization, fee collection, transportation, and exam conduction. It integrates payment gateways (Worldline, Razorpay, ATOM) for online transactions and Google tools (Drive, Calendar, YouTube) for study material and online lectures. Teachers can create custom exams, track student progress, and conduct secure online assessments with tab-change validation. Designed for both online and offline schools, its desktop version ensures functionality without continuous internet, with daily data sync. It provides seamless administration and efficient learning.

Technologies: ASP.NET, .NET Core 3.1, Flutter, C#, MSSQL, SQLite3, using ADO.NET for web and Dapper for mobile

QMS

QMS (Quality Management System) is a hospital domain software designed to streamline audit processes and issue management within healthcare facilities. It ensures comprehensive tracking of audits, allowing hospitals to monitor compliance, operational efficiency, and issue resolution. Through QMS, users can conduct audits, document findings, track corrective actions, and generate structured reports for regulatory and internal analysis. The software enables healthcare organizations to maintain quality standards, optimize processes, and ensure patient safety by systematically addressing issues raised during audits. QMS offers a scalable, secure, and intuitive interface, allowing seamless data management and report generation for healthcare professionals and administrators.

Technologies: Built with .NET Core 3.1, Entity Framework, VueJS, and MSSQL

Enterprise App

Enterprise App is a mobile-based accommodation management system designed for individuals seeking PG (Paying Guest) housing when visiting a new city. It streamlines the rental process, allowing customers to search, book, and manage rooms or shared spaces with ease. The app caters to multiple user roles, including Admin, Customers, Agents, Electricians, and Maintenance Staff, ensuring seamless property management, tenant services, and maintenance requests. Customers can browse available listings, book accommodations, and raise service requests, while agents assist in finding suitable housing and maintenance providers handle infrastructure repairs. The Enterprise App provides an efficient, scalable, and user-friendly experience, optimizing the housing rental process for both newcomers and property managers.

Technologies: Flutter, .NET Core 3.1, Dapper
